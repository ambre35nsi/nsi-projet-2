Pour lancer le jeu :
- ouvrir le fichier index.html
- cliquer sur le bouton Commencer

Les options :
- bouton pour changer de carte
- bouton pour changer de niveau

Les régles :
- Pacman doit manger toutes les pastilles sans être attraper par un fantôme.
- Le joueur marque 10 points pour chaque pastilles mangées
- Si pacman mange toutes les pastilles, le joueur a gagné
- Si pacman est touché par un fantôme avant d'avoir mangé toutes les pastilles, le joueur a perdu
